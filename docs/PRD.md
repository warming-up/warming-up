# PRD - 워밍업 (WarmUp)

## 한 줄 소개

> **"몇 시에 출발해야 하지?"가 아니라 "몇 시부터 준비해야 하지?"를 알려주는 준비 시간 역산 앱**

도착 시간을 기준으로 준비 시작 시간을 역산해 알려주는 알람 앱

---

# 문제

## 페인포인트

- 언제 출발할지는 알지만, 언제부터 준비해야 하는지는 모른다.
- 일정마다 준비 시간이 달라 매번 감으로 계산한다.
- 준비물을 빠뜨리거나 준비가 길어져 출발 시간이 밀린다.

지도 앱은 이동 시간만 계산한다.

하지만 실제 지각은 이동보다 **외출 준비 시간** 때문에 발생한다.

---

# 해결 방법

워밍업은 도착 시간을 기준으로 준비 과정을 역산한다.

### 입력

- 도착 시간
- 이동 시간
- 도착 여유 시간
- 준비 루틴

### 출력

- 준비 시작 시간
- 집에서 나갈 시간
- 준비 타임라인
- 체크리스트

즉,

> **"몇 시에 출발할까?"가 아니라 "몇 시부터 준비해야 할까?"를 알려준다.**

---

## 준비 시간 역산

```text
도착 시간
    ↑
도착 여유 시간
    ↑
이동 시간
    ↑
준비 루틴
    ↑
준비 시작 시간
```

사용자는 상황별 루틴(출근, 운동, 면접 등)을 저장해두고,

도착 시간만 입력하면 다음 정보를 확인할 수 있다.

- 준비 시작 시간
- 집에서 나갈 시간
- 준비 타임라인
- 체크리스트

---

# 핵심 기능 (MVP)

## 1. 루틴 생성

예시)

### 면접

- 씻기 (20분)
- 머리 말리기 (10분)
- 정장 입기 (10분)
- 서류 챙기기 (5분)

---

## 2. 일정 실행

### 입력

- 도착 시간
- 이동 시간
- 도착 여유 시간
- 사용할 루틴

---

## 3. 준비 시간 역산

```text
준비 시작 시간 =
도착 시간
- 도착 여유 시간
- 이동 시간
- 준비 시간
```

### 결과

- 준비 시작 시간
- 집에서 나갈 시간
- 단계별 타임라인

---

## 4. 체크리스트

예시)

- [ ] 신분증
- [ ] 노트북
- [ ] 충전기

---

## 5. 알림 (시간 여유 시)

- 준비 시작 알림
- 집에서 나가기 알림

---

# 차별점

| 지도 앱      | 워밍업              |
|-----------|------------------|
| 출발 시간 계산  | 준비 시작 시간 계산      |
| 이동 시간만 고려 | 준비 루틴 + 이동 시간 고려 |
| 길 안내      | 외출 준비 전체 관리      |
